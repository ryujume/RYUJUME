# 류주메

#### 개요

류주메는 대마고에서 주말에도 잔류하는 권하경, 김다은, 이채은이 만드는 레주메 관리 서비스입니다.

웹과 ios플랫폼으로 개발할 예정입니다.



#### 기능명세

1. 회원가입
   1. 아이디
   2. 비밀번호
   3. 이름
2. 로그인
   1. 아이디
   2. 비밀번호
3. 레주메
   1. 작성
      1. 개인정보 (필수)
         1. 이름
         2. 사진
         3. 소속
         4. 연락처
         5. 이메일
      2. 추가하기
         1. 항목
         2. 상세
   2. 수정
